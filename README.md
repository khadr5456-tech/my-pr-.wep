Personal Portfolio Website

This is a fully responsive personal portfolio website built with HTML, CSS, and Bootstrap, designed to showcase a developer's skills, experience, and projects. It features a modern, clean, and professional design, suitable for personal branding, freelance portfolios, or job applications.

Sections

Home / Hero Section – A visually appealing introduction with a background image, overlay, and animated title to grab visitors' attention.

About Me – Displays personal details including name, profile, email, phone, skills, and a profile picture. Skills are shown using progress bars to represent proficiency in technologies like HTML, CSS3, JavaScript, and PHP.

Services – Lists offered services such as Web Design, Web Development, Photography, Responsive Design, Graphic Design, and Marketing, with clear icons and descriptions.

Portfolio / Work – Showcases past projects with interactive image galleries, lightbox functionality, and links to project details. Each project includes category and date information.

Blog – Highlights recent blog posts with images, categories, short descriptions, authors, and timestamps.

Testimonials – A slider section displaying client feedback, including names, profile images, and text.

Contact – Provides a functional contact form, address, phone number, email, and social media links. Users can send messages directly via the form.

Footer – Includes copyright information and design credits.

Features

Parallax backgrounds and overlays for visual depth.

Smooth scrolling navigation and dropdown menus.

Animated counters to showcase achievements.

Responsive layout for desktops, tablets, and mobile devices.

Interactive elements such as lightboxes, sliders, and hover effects.

How to Run Locally

Download all project files and keep them in the same folder.

Open index.html in a modern web browser like Chrome, Edge, or Firefox.

For full functionality, use a local server:

Open the project in VS Code.

Install the Live Server extension.

Right-click index.html → Open with Live Server.

Make sure all image paths, CSS, and JS files are correct.

Optional: To deploy online, upload the project to a GitHub repository and enable GitHub Pages.
